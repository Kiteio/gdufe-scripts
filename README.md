# 广东财经大学脚本集

> **请勿滥用 概不负责**

我们将基于 [template.py](template.py) 制作一系列模块，各个模块均相互独立存放于一个 ***.py*** 文件中，方便使用。

项目提供了 [template.py](template.py) 供二次开发，相关的依赖放置于 [requirements.txt](requirements.txt)。

## 结构
```
gdufe-scripts
├── course               # 选课脚本（TODO）
├── evaluation           # 教学评价脚本
├── network              # 校园网登录脚本
├── requiremenet.txt     # template.py 依赖库
└── template.py          # 教务系统脚本模板文件
```
